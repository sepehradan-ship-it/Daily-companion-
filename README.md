# Daily Companion 📅

A comprehensive daily management app that combines task lists, journaling, planning, and reminders in one beautiful interface.

## Features

### 📝 Tasks
- Create and manage daily to-do items
- Mark tasks as complete
- Track progress with a percentage indicator
- Delete tasks you no longer need
- All tasks automatically saved to your browser

### 📔 Journal
- Write and save daily journal entries
- One entry per day (entries are date-based)
- Write reflections, thoughts, and feelings
- Clear entries when needed
- Automatically saves to browser storage

### 🗓️ Planner
- Schedule events throughout the day
- Organize by time slots
- Events automatically sorted by time
- Delete events you want to remove
- Keep track of your daily schedule

### 🔔 Reminders
- Set time-based reminders for daily tasks
- Browser notifications for reminders (with permission)
- Fallback alert notifications
- Automatically checks for reminders every minute
- Organized by time for easy reference

## How to Use

1. **Open** `index.html` in your web browser
2. **Click** the tab buttons to switch between features
3. **Add** items using the input fields and buttons
4. **Manage** your items by checking off tasks, deleting items, or updating entries
5. **All data** is automatically saved to your browser's local storage

## Data Storage

All your data is stored locally in your browser using localStorage. This means:
- ✅ Your data is private and never leaves your computer
- ✅ Data persists even after closing the browser
- ✅ Each day gets its own journal entry
- ⚠️ Clearing browser data will delete stored entries

## Browser Compatibility

Works with all modern browsers including:
- Chrome/Edge
- Firefox
- Safari
- Opera

## Tips

- **Tasks**: Use the progress bar to stay motivated!
- **Journal**: Write daily for personal growth and reflection
- **Planner**: Plan your day ahead for better productivity
- **Reminders**: Set time reminders to stay on track with your goals
